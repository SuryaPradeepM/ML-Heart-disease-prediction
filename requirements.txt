
The folder "Final-Project" contains code for project, one report and some images for plots generated by the program.

To Run the Program:

1. Ensure that python 3, Anaconda, Jupyter Notebook is installed in your machine and other dependencies namely numpy, matplotlib, seaborn and sklearn libraries 
2. Open Jupyter Notebook.
3. run the ipynb files by changing to that directory and run the files there.
4. Change “ matplotlib.use(Agg) to matplotlib.use(TKAgg) if you want to see the plot in Mac.
