
The folder "Final-Project" contains code for project, report and images of plots generated by the program.

To Run the Program:

1. Ensure that python 3, Anaconda, Jupyter Notebook are installed in your machine and other 
   dependencies namely numpy, matplotlib, pandas, seaborn and sklearn libraries with newest
   versions.
2. Open Jupyter Notebook. (Open it in this path by updating path to jupyter)
3. Run the ipynb files by changing to that directory and run the files there.
4. Change “ matplotlib.use(Agg) to matplotlib.use(TKAgg) if you want to see the plot in Mac.
